# Higher Programming Lessons - Python and MySQL

==== how to fireup mysql: ==========

-> sudo service mysql start
-> sudo mysql

==== change user and password: =====

show users:
-> SELECT user from mysql.user;

alter user to set password (for root user):
-> ALTER USER root@localhost IDENTIFIED BY 'password' (check if it worked)

==== Add a table using from an SQL script =======
cat 0-select_states.sql | sudo mysql -uroot -p


=== Run a python script to get mysql data ===
sudo python_file.py root password database_name
